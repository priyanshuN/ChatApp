# ChatApp

## How to commit
- git clone https://github.com/priyanshuN/ChatApp.git
- cd ChatApp
- Create a branch
    - git branch *branch_name*
    - git checkout *branch_name*
- To avoid conflict use:
    - git pull origin master
- git add .
- git commit -m"*commit message*"
- git push origin *branch_name*
